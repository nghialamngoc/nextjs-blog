<H2 id="pre-rendering">1. Pre-rendering</H2>

Two forms of pre-rendering:

- SSG - Static Site Generation
- SSR - Server Side Rendering

Pre-rendering (Using Next.js)

![NextJs Pre-rendered](https://nextjs.org/static/images/learn/data-fetching/pre-rendering.png)

<H2 id="ssg-static-site-generation">2. SSG - Static Site Generation</H2>

The HTML is generated at the **build-time** and is reused for each request

![Static Generation](https://nextjs.org/static/images/learn/data-fetching/static-generation.png)

<H2 id="ssr-server-side-rendering">3. SSR - Server Side Rendering</H2>

The HTML is generated on **each request**

![Server Side Rendering](https://nextjs.org/static/images/learn/data-fetching/server-side-rendering.png)
